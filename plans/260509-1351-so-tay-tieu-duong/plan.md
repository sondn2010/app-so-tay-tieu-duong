### File Ownership (CRITICAL вҖ” no overlap)

| Agent | Owns | Must NOT touch |
|-------|------|----------------|
| Coordinator | `src/lib/core/**`, `src/lib/app.dart`, `src/lib/main.dart` | Feature folders during parallel phase |
| Agent 1 | `src/lib/features/blood_sugar/**` | core/, medication/, knowledge/ |
| Agent 2 | `src/lib/features/medication/**` | core/, blood_sugar/, knowledge/ |
| Agent 3 | `src/lib/features/knowledge/**`, `src/assets/knowledge/**` | core/, blood_sugar/, medication/ |

> All Isar schemas are created by **Coordinator in Phase 02** вҖ” agents only implement repository/providers/screens on top of pre-existing schemas.

### Sync Points

1. **After Phase 01** вҖ” Coordinator verifies `flutter pub get` succeeds before Phase 02
2. **After Phase 02** вҖ” Coordinator verifies `build_runner` succeeds + bottom nav shell renders before spawning agents
3. **After all 3 parallel** вҖ” Coordinator collects DONE status from all agents before Phase 06
4. **After Phase 06** вҖ” Final `flutter analyze` + `flutter build apk --release`
